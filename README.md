# 🗄️ SQL Retail & Sales Analysis Project  

This repository documents my SQL learning journey during the **Level 3 Data Technician Bootcamp**.  
Throughout the week, I learned how to query structured data, design relational databases, and extract insights from real retail, sales, and global datasets.

This project reflects not just the technical skills I gained, but also how my understanding of data analysis developed.

---

## 🧠 What I Learned

### 🔍 Core SQL Querying
I learned how to retrieve and filter data using:

- `SELECT`  
- `WHERE`  
- `ORDER BY`  
- `DISTINCT`  
- `IN`, `BETWEEN`, `LIKE`  
- `LIMIT`

### 📊 Aggregation & Grouping
I practised summarising data using:

- `GROUP BY`

### 🔗 JOINs & Relational Thinking
Understanding JOINs was one of the biggest turning points in my learning.  
I used:

- `INNER JOIN`  
- `LEFT JOIN`  
- `RIGHT JOIN`  
- `FULL JOIN`  
- `CROSS JOIN`

These helped me combine tables to answer questions like:

- Which supplier provides each product  
- Which learners are enrolled on which courses  
- Which courses have no enrolments  
- Full customer + order + employee reports  

This taught me how relational databases prevent duplication and keep data consistent.

### 🧱 Database Design & Normalisation

I used SQL to explore real retail scenarios, including:

- Recent order analysis

I analysed global data to answer questions such as:

- Population ranges

This helped me practise writing queries that support real‑world decision‑making.

### ✔ Scenario‑Based JOIN Activities
I applied JOINs to solve problems like:

These exercises strengthened my understanding of how relational data works.
